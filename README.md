# Löwenherz PWA

SMALL-Reminder Tracker — Begleit-App zum Buch "Löwenherz: Angst wird erlernt. Gelassenheit auch."

## Deploy

Vanilla HTML/CSS/JS — kein Build-Schritt nötig. Direkt auf Vercel deploybar:
1. Repository importieren
2. Framework Preset: Other
3. Deploy

## Tech Stack
- Vanilla JavaScript (ES6 Modules)
- IndexedDB (alle Daten lokal)
- Service Worker (Offline-fähig)
- PWA (installierbar auf iOS/Android)
